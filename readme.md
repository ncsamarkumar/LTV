<h1>Extracting Recalls Of Vehicle Identification Number (VIN)</h1>

Files:
1. ford.py : Program to get the details of VIN ID from the ford website
2. chevrolet.py : Program to get the details of VIN ID from the chevrolet website
3. utils.py : Program to create the output folder and store all the output in their respective csv file
4. constants.py : Program which has those part of the code which are common in all scripts

To Execute Program for Ford:
1. Give this command in terminal "Python ford.py "Ford__Chevrolet_VINs.xlsx"

To Execute program for Chevrolet:
1. Give this command in terminal "Python chevrolet.py "Ford__Chevrolet_VINs.xlsx"

All the output csv files will be created in the output folder which will be created automatically if not present in the current directory.
FordRecall.csv : Will have all the recalls related to the Ford VIN_ID
FordCustomerSatisfaction.csv : Will have the reviews give by the customers
ChevroletRecall.csv : Will have all the recalls related to the Chevrolet VIN_ID
Ford__Chevrolet_VINs.xlsx: Have all the VIN_ID for which the code will get the details from the website, Don't put this inside the output folder.
